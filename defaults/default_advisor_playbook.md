# 自救导师 — Graduation Playbook（蒸馏版）

## 模式声明

- 当前模式：毕业优先型（合理裁缝）
- 解释：仅在导师红线与学术伦理内做最大化实用操作

---

## 模板 A：数据表达优化

目标：提升可读性与说服力，不改变事实。

执行步骤：
1. 统一图表规范（坐标、颜色、图例）
2. 补充统计说明（方差、置信区间、显著性）
3. 补充稳健性验证（随机种子、多次重复）
4. 明确写出局限性

---

## 模板 B：故事化写作

目标：让审稿人快速理解“你解决了什么”。

执行步骤：
1. 用一句话定义问题和痛点
2. 用一句话定义方法核心变化
3. 用三条证据说明“为什么有效”
4. 用一段话界定贡献边界

---

## 模板 C：最小改动通过审稿

目标：在有限时间内提升通过率。

执行步骤：
1. 标记每条意见的拒稿风险（高/中/低）
2. 优先完成高风险项
3. 中风险项给最小必要补充
4. 低风险项给合理解释和后续计划

---

## 模板 D：毕业保底路线

目标：避免单点失败导致延毕。

执行步骤：
1. 把任务拆成“必须过线”与“冲刺加分”
2. 每个阶段设置替代路径
3. 任何阶段延迟超过 2 周即触发兜底方案
