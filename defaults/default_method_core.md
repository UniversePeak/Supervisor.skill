# 自救导师 — Method Core（预蒸馏通用方法论）

## 适用目标

- 用户素材不足：还原不了完整导师方法
- 导师能力不足：只给压力不给方法
- 时间紧迫：需要先用稳定框架保住毕业主线

---

## 任务分解框架（统一使用）

1. 定义问题：目标、约束、最晚截止
2. 拆分任务：必须做 / 可选做 / 放弃项
3. 设验收：每项有可量化完成标准
4. 风险分级：高/中/低 + 对应兜底路径
5. 周复盘：保留、降级、切换三选一

---

## 决策优先级

1. 学术红线与合规
2. 毕业节点与通过概率
3. 可复现与可交付
4. 上限优化与长期价值

---

## 默认策略建议

- 素材充分度 < 40：`template_first`
- 素材充分度 40-70：`hybrid_distill`
- 素材充分度 > 70：`strict_distill`

---

## 输出格式（强制）

1. 当前判断（你现在在哪个阶段）
2. 本周三步（只给 1-3 步）
3. 每步验收标准（量化）
4. 风险与备选方案（失败时切哪条线）
