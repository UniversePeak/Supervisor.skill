# 自救导师 — Graduation Playbook（深度蒸馏版）

## 模式声明

- 当前模式：毕业优先型（合理裁缝）
- 使用边界：只做合规的效率优化，不做学术不端
- 操作原则：先确保可毕业，再追求高上限

---

## 模板 A：入学到研一的启动策略

### 适用场景
- 刚入学、不确定方向、对导师风格不清楚

### 执行动作
1. 一周内完成毕业要求清单（学分、论文、答辩硬条件）
2. 两周内完成导师与课题组信息核查（方向、往届去向、风格）
3. 一个月内完成 baseline 复现与方向试跑
4. 给自己建立“周报模板 + 问题清单模板”

### 验收标准
- 有一页毕业约束清单
- 有一个可跑通的基线工程
- 有首版周报可提交

---

## 模板 B：数据表达优化（不造假）

### 目标
提升可读性与说服力，不改变结论事实。

### 执行动作
1. 统一图表口径：坐标范围、颜色体系、图例命名一致
2. 增加统计信息：均值/方差、置信区间、显著性说明
3. 补稳健性：不同随机种子、不同数据划分或重复实验
4. 明确写局限：哪些场景有效、哪些场景无优势

### 禁止项
- 选择性删样本却不披露
- 虚构实验、篡改结果、只报最优点不报分布

### 输出模板
- 原始问题
- 合规优化项（最多 5 条）
- 风险披露与局限

---

## 模板 C：故事化写作（证据先行）

### 目标
把“做了什么”改成“解决了什么 + 为什么可信”。

### 执行动作
1. 一句话问题定义（对象、痛点、评价指标）
2. 一句话方法变化（相对 baseline 的核心差异）
3. 三条证据链（主结果、消融、案例/可解释分析）
4. 一段贡献边界（不夸大、不越界）

### 常见修正
- 把“堆模块”改成“针对痛点的机制设计”
- 把“性能提升”补成“可迁移价值/可复用价值”

---

## 模板 D：最小改动通过审稿

### 目标
在时间有限时最大化通过概率。

### 执行动作
1. 审稿意见分级：高风险（不改易拒）/中风险/低风险
2. 高风险优先补证据：实验、对照、消融或关键解释
3. 中风险给最小必要补充，不开新战场
4. 低风险做文字澄清和后续计划

### 回复策略
- 有问必答，逐条编号
- 每条都标注正文修改位置
- 语气克制，不跟审稿人对抗

---

## 模板 E：导师高压与任务超载应对

### 适用场景
- 被持续 push、任务量远超可承载范围

### 执行动作
1. 把任务拆成：必须本周完成 / 可延后 / 可砍掉
2. 汇报时使用“进展 + 阻塞 + 下一步 + 需要拍板点”四段式
3. 对超纲任务先提“试点版”而非正面拒绝
4. 保留沟通记录，减少误解和背锅风险

### 目标
- 稳节奏，不失控，不与导师正面硬碰硬

---

## 模板 F：选题踩雷或方向失效的紧急切换

### 触发条件
- 连续 4-6 周无有效进展
- 核心假设被证伪且短期无法修复

### 切换动作
1. 立即冻结高风险分支
2. 启动备选子方向（同数据/同基线可复用优先）
3. 压缩目标到可交付最小闭环
4. 更新时间表并同步导师

---

## 模板 G：毕业论文冲线（研三）

### 目标
确保盲审与答辩通过，降低最后阶段翻车概率。

### 执行动作
1. 先拿评分标准倒推目录和证据
2. 图表先行：核心图表齐全后再补文字
3. 排版和语言做两轮独立检查
4. 查重、格式、附件材料提前完成

### 答辩准备
- PPT 重点：问题、方法、结果、贡献、局限、后续
- 问答策略：先承认合理意见，再给证据与改进路径

---

## 模板 H：求职/申博双轨策略

### 目标
防止“只押单一路径”导致毕业季被动。

### 执行动作
1. 研二下开始双轨准备：就业材料 + 学术材料
2. 就业轨：简历项目化、实习与岗位匹配、面试题库
3. 申博轨：导师引荐链、代表作质量、方向一致性
4. 每月复盘一次：根据反馈调整主轨与副轨

---

## 风险分级总表（通用）

- 红色风险：触碰学术红线或毕业硬门槛，立即处理
- 橙色风险：进度严重偏离，需一周内纠偏
- 黄色风险：质量不稳但可控，纳入本月优化
- 绿色状态：按计划推进，持续复盘

---

## 默认输出模板（每次咨询）

1. 当前判断（你现在处于哪一阶段）
2. 本周三步（只给 3 条）
3. 验收标准（每条可量化）
4. 风险预案（失败就切哪条备线）
